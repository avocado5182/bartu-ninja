Projects

gwa run

gwa run is a mobile endless runner game featuring a shop along with various skins! Using the in-game currency, users can buy skins for their character and use them to customize their experience. The game is built using the Unity Game Engine in C#. It released on May 31st, 2022 to the Google Play Store. The game has a 5-star rating.

Manic Island Bot

The Manic Island bot is a bot coded in JavaScript for the Discord social messaging platform. It features various commands for moderation, utility, and fun. It also features an economy system featuring a shop and jobs for users to earn a virtual currency. As users work, they gain experience, which can be used to level up and increase their salary. The bot is no longer in development. It was started in November of 2020.

Complexius

Complexius is a bullet-hell style game made for the 2022.1 Brackeys Game Jam in 1 week, from February 21st to February 27th. In the game, you work your way through an 8-level gauntlet, slaying enemies and weaving your way through the endless barrage of bullets. The user utilizes a variety of weapons to defeat the final boss who is bound on destroying the complex plane: Quaternius, devourer of dimensions. Complexius was built in the Unity Game Engine using C#. It can be found on itch.io.
